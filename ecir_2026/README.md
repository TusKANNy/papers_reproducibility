## Reproduction Instructions

To reproduce the results from this paper:

1. Unzip the archive in this folder:
   ```bash
   unzip ecir2026_multivec.zip
   ```

2. Follow the instructions in the `REPRODUCTION_GUIDE.md` file contained within the unzipped repository.
